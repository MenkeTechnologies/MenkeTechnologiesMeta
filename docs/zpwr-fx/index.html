>_JUCE MODULAR MULTI-EFFECTS

A modular patch graph of audio effects — free cabling, not a fixed slot rack. Paid product.

Architecture

zpwr-fx is the audio host for the shared zpwr-patch-core engine: a free-routed graph of modules (cables by source id, fan-out, feedback resolved with a one-sample delay) rather than a fixed slot chain. Every node param carries a (source, depth) entry in the mod matrix; live edits are lock-free with an atomic graph swap on structural changes; whole patches serialise to JSON with versioning + source-id migration.

ext In L/R  →  [ Filter ]  →  [ Delay ]  →  …  →  Out
                 ↑ mod matrix      ↑ per-cable gain + colour
   (any node → any node; feedback allowed)

Each layer is a full engine copy (layers are unlimited); the ⚡ EZ WIRE mode auto-chains In → blocks → Out so linear hosts never touch a cable. The WebView UI and its expandable soft-knob pool come from zpc::WebEditor.

Module catalog

250+ audio/synth module types in the shared registry (buildFxRegistry / registerAudioModules) — dynamics, EQ/filter, delay, reverb, modulation, distortion/saturation, pitch, spectral (FFT), stereo, lo-fi, and creative/glitch families (plus 54 note-stream modules in zpwr-midi-fx — 300+ DSP blocks across the stack). Effect-by-effect parity against every major DAW + plugin catalog is tracked in FX_PARITY.md.

Build your own — primitives + code

Not just presets: wire the 250+ modules into your own algorithm (any node to any node, feedback, mod matrix), or write per-sample DSP in the Expr code module — a sandboxed VM with 8 automatable knobs, persistent state, ~30 math functions, and a fractional tap() delay line for hand-built filters/combs/delays. UAD and Eventide native plugins ship fixed emulations; the Bitwig Grid has primitives but no scripting node — zpwr-fx has both.

Analog models

A dedicated pack of 63 named-circuit analog models (registerAnalog) — faithful generic topologies, not sample/IR clones:

  • Synth filters (7) — Minimoog, Jupiter-8, MS-20, Oberheim SEM, EMS VCS3, EDP Wasp, TB-303.
  • Compressors & limiters (9) — Fairchild, LA-2A, LA-3A, 1176, dbx 160, SSL bus, API 2500, Distressor, Sta-Level.
  • EQs (8) — Pultec EQP-1A, API 550, Neve 1073, SSL E, SSL G, Manley Massive Passive, Helios Type 69, GML 8200.
  • Preamps & tape (6) — Neve 1073 pre, API 312 pre, tube console pre, SSL bus glue, Studer A800, Ampex ATR-102.
  • Distortion / pedals (12) — Tube Screamer, RAT, Big Muff, Fuzz Face, Klon, DS-1, MXR Distortion+, Octavia, Super-Fuzz, Rangemaster, OCD, Metal Zone.
  • Guitar amps (5) — Fender Tweed, Blackface, Marshall Plexi, Vox AC30, Mesa Rectifier.
  • Enhancers (2) — Aphex Aural Exciter, BBE Sonic Maximizer.
  • Phasers / tremolo (5) — MXR Phase 90, EHX Small Stone, Uni-Vibe; Fender optical + tube-bias tremolo.
  • Chorus / ensemble (3) — Roland Dimension D, Boss CE-1, string ensemble.
  • Reverb & echo units (6) — EMT 140 plate, echo chamber, AKG BX-20 spring; Roland RE-201 Space Echo, Binson Echorec, EHX Memory Man.
