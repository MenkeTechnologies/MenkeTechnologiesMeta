zpwr-fx

JUCE modular multi-effects plugin · MenkeTechnologies · paid product

One unit, a rack of effect slots, every effect category. JUCE's parameter model (AudioProcessorValueTreeState) is static — the host needs the full parameter list at construction time. So the rack is a fixed set of ordered slots, each with a user-selectable effect type and a bank of generic macro parameters, which keeps host automation stable.

Input → [ Slot 0 ] → [ Slot 1 ] → … → [ Slot 7 ] → Output
          type=Filter   type=Delay        type=None
          bypass/mix     bypass/mix        (passthrough)
