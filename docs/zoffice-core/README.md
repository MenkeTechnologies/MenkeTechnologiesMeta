### `[THE EMBEDDABLE OFFICE ENGINE]`

> *"One office engine, every app."*

**zoffice-core** is the pure-Rust embeddable engine behind [`zoffice`](https://github.com/MenkeTechnologies/zoffice) — document / spreadsheet / presentation parsing + editing, no GUI dependencies. Links natively into Rust/Tauri hosts and over a C ABI elsewhere. Created by MenkeTechnologies.
