ZPDF // THE CYBERPUNK PDF EDITOR

// Tauri v2 desktop editor over the embeddable pure-Rust zpdf-core engine — built to replace Adobe Acrobat
Engineering Report Port Report

zpdf is the GUI: a Tauri v2 desktop PDF editor aiming to cover the union of what Adobe Acrobat Pro and macOS Preview do — viewing, page management, text/object editing, annotation, forms, signatures, security, redaction, OCR, convert/export. It sits on zpdf-core, the embeddable pure-Rust engine (lopdf, MIT) that holds all the logic — so the same code that powers this app also embeds into the other MenkeTechnologies apps (traderview, …).

App = GUI

zpdf is the desktop editor. All PDF logic lives in the zpdf-core submodule; the app is the interface.

Core = embeddable

zpdf-core is the engine/logic — embeddable into any host so other apps mark up PDFs in-window.

Honest port report

The port report derives status from zpdf-core source — a feature is "done" only with real engine code behind it.

Status

The engine is real and tested; the Tauri GUI shell is in progress. Track exactly what the engine implements against the full Acrobat/Preview surface in the feature port report, and see module/test detail in the engineering report.
