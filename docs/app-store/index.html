>_MENKETECHNOLOGIES APP STORE

A static storefront for the MenkeTechnologies audio products. No build step — pure HTML / CSS / JS.

Catalog

Sells the audio products: Audio-Haxor (Tauri/JUCE plugin scanner), traderview (self-hosted trading journal), zpwr-synth (JUCE synthesizer), and zpwr-fx (JUCE multi-effect). The product catalog is a single source of truth in store.js.

Design

Shares the strykelang-docs HUD / cyberpunk design system (hud-static.css, tutorial.css, hud-theme.js): Orbitron + Share Tech Mono, CRT scanlines, neon borders, and five swappable color schemes (Cyberpunk, Midnight, Matrix, Ember, Arctic) plus light / dark — the same visual language as these docs.

Pages

  • index.html — storefront: hero, search, category filters, product grid
  • product.html — product detail page (reads ?id=<product>)
  • checkout.html — checkout: express wallets, card form, summary
