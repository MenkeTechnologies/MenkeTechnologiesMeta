The paper trail for the stack.

The companion books, reference manuals, the zpwr encyclopedia, and four original novels for the stryke / zshrs / zpwr stack. The reference content is generated from the public source; the novels are hand-written. Everything shares one cyberpunk pandoc → lualatex pipeline — 12 typeset deliverables, 2,785 pages. See the engineering report for the full breakdown.
