What zoffice is

A from-scratch office suite in Rust covering the office triad — documents, spreadsheets, and presentations — packaged as a Tauri v2 desktop app behind a cyberpunk HUD, built to replace Microsoft Office. The editing engine is factored out as zoffice-core so the same engine can be embedded inside the other MenkeTechnologies apps rather than living only in the desktop binary. The suite is designed to interoperate with the Microsoft Office document, spreadsheet, and presentation formats.

The split between the desktop app (zoffice) and the engine (zoffice-core) is the load-bearing architectural decision: the engine carries the document model and editing operations with no GUI dependency, so the same code can power both the desktop suite and embeds in the other MenkeTechnologies apps.
