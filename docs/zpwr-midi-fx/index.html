>_JUCE MIDI-EFFECTS PLUGIN

The MIDI-domain companion to zpwr-fx. New project — scaffolding stage.

Concept

Where zpwr-fx processes the audio stream, zpwr-midi-fx transforms the MIDI stream before it reaches an instrument — arpeggiation, chord generation, scale quantization, velocity / timing shaping, and note remapping — using the same fixed-slot rack model so host automation stays stable.

MIDI in → [ Slot 0 ] → [ Slot 1 ] → … → [ Slot N ] → MIDI out
            type=Arp      type=Chord        type=None
            bypass         bypass            (passthrough)

Formats

VST3AUCLAPStandalone

Registers as a MIDI-effect plugin where the host supports the category (Logic AU MIDI FX, CLAP note-port plugins).

Scaffolding stage — this page is the project template. Engine, parameter tree, and slot catalog land as the JUCE wrapper is built out.
