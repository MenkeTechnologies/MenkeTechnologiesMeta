Each MenkeTechnologies app or engine ports the feature surface of an existing reference tool. A port report enumerates that surface and tracks status per feature, derived from source — a feature counts as done only when a real, non-stub symbol exists. This page links every report. Counts in the Coverage column are the report's own feature/module totals.
