⚠ Read this first — what this report is, and isn't

This is a citation-backed coverage map and roadmap: every PORTED row names the exact file:line (or file + function) in this repo where the feature lives — the citation is the proof, not a self-assigned score. There is no ‘100% / 0 gaps’ claim — the opposite: most of Premiere is GAP (real, in-scope, not yet built). What's PORTED today is the video-clip foundation (Phase 1+2): clips on the timeline, filmstrip, in/out trim, the program monitor + pop-out player, and the multi-track timeline editing inherited from the DAW arranger.
