Docs GitHub
Tool coverage: 532 of 578 in-scope tools implemented (92%). Every toolbar tool and major menu feature of Adobe Photoshop, Adobe Illustrator and GIMP is listed below and mapped to the zphoto-core engine command that implements it. Across the three apps: 532 covered 32 partial 14 missing 24 N/A (out of scope) — 602 rows total. Verified against zphoto's live command surface (479 commands) and the source-derived port checklists.

zphoto ports the union of Photoshop (raster), Illustrator (vector) and GIMP onto one shared layer stack, driven through a single Engine::invoke(cmd, args) surface. This page is the exhaustive cross-reference: pick any tool from any of the three apps and see the exact zphoto command behind it. N/A marks features intentionally out of scope for a headless engine — live-only GUI navigation (Hand/Zoom/Perspective Grid), web-export slices, CMYK/prepress, AI neural selection, and live-UI plugins (GIMPressionist, Gfig).
