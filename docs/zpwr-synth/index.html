>_JUCE SOFTWARE SYNTHESIZER

Semi-modular instrument modeled on SynthMaster 3 (KV331 Audio). Paid product.

Overview

The DSP core lives in dsp/ as a pure-C++ static library with no JUCE dependency, so it is unit-testable headless. The JUCE wrapper owns the parameter tree and feeds the engine.

Architecture

  • WavetableBank — shared, band-limited, procedurally built
  • Layer[16] — SynthMaster 3 supports 16 layers per instrument
  • Polyphonic voice pool with round-robin voice stealing
  • Generators: basic PolyBLEP oscillator + virtual-analog oscillator (saw/pulse, PWM, unison, drift)

Formats

AUVST3Standalone

macOS (Apple Silicon / Intel) and Linux (x86_64 / aarch64).
