Fully modular instrument modeled on SynthMaster 3 (KV331 Audio). Paid product.

Overview

A fully modular instrument built on the shared zpwr-patch-core engine: each voice is a free patch graph of modular blocks (not a fixed signal path), so you wire oscillators, filters, envelopes and modulators into your own voice. The pure-C++ DSP core (dsp/) has no JUCE dependency and is headless-unit-testable; the JUCE wrapper owns the parameter tree.

Architecture

  • Instrument → Layer[16] → Voice — up to 16 stacked layers, each a polyphonic voice pool over its own patch graph.
  • 45 voice modules (buildSynthRegistry) — virtual-analog, wavetable, FM, additive, supersaw, Karplus and sampler oscillators; filters (SynthFilter / diode ladder); ADSR / LFO / S&H modulators; VCA / mixer / scaler.
  • FX-bus rack — a master bus plus 2 aux buses, each a patch graph running the full shared 900+-module audio registry (the same pack as zpwr-fx, including the 100+ analog models).
  • Per-param mod matrix, sidechain + global mods into the bus rack, and the shared zpc::WebEditor WebView UI.

Voice modules

  • Filters — Filter (TPT state-variable, 4 modes) and DiodeLadder (4-pole, TB-303 grit, drive).
  • Shapers — Folder (west-coast wavefolder), Waveshaper (4-curve), Crusher (bit + sample-rate reduction).
  • Modulators — Env (ADSR), LFO, StepLFO (stair / random S&H), NoiseLFO (coloured-noise CV), SampleHold, Glide (portamento), Scaler (scale/offset/curve a CV).
  • Routing / amp — VCA, Mixer, Gain, RingMod, Drive.

Plus the FX-bus rack: a master bus + 2 aux buses, each a patch graph running the full shared 900+-module audio registry (same pack as zpwr-fx, incl. the 100+ analog models). Full node reference: reference.html.

Modulation, soft keys & MPE

Modulation is the patching: any LFO/Env output or soft key patched into a node input — or mod-routed to a param — is a mod-matrix connection. Host-automatable parameters are the soft keys (expandable pool, 8 active by default, active count saved in plugin state) plus master in/out/bypass; everything else lives in the patch JSON. External per-voice sources include Note, Gate, Velocity, Mod Wheel, Pitch Bend, Aftertouch and MPE pressure / slide / bend.

Presets & UI

A full Bank A of 128 factory voices ships in category-prefixed names (BA bass, LD lead, PD pad, KY keys, PL pluck, DR perc, SEQ/FX) spanning subtractive, FM, additive, supersaw, wavetable, vector, sync and Karplus voices — each loads playable with three named soft-knob macros and mod-wheel / velocity / aftertouch routed. User presets save as .zsynthpatch.

The editor is the shared zpc::WebEditor WebView patcher — drag-to-patch neon cables (fan-out, feedback, per-cable gain + colour), a node grid, per-param mod matrix, and an ⚡ EZ MODE that lays down a playable generators → VCA → Filter → out voice for linear use. An on-screen + computer keyboard plays from the editor.
