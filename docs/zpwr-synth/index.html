>_JUCE SOFTWARE SYNTHESIZER

Fully modular instrument modeled on SynthMaster 3 (KV331 Audio). Paid product.

Overview

A fully modular instrument built on the shared zpwr-patch-core engine: each voice is a free patch graph of modular blocks (not a fixed signal path), so you wire oscillators, filters, envelopes and modulators into your own voice. The pure-C++ DSP core (dsp/) has no JUCE dependency and is headless-unit-testable; the JUCE wrapper owns the parameter tree.

Architecture

  • Instrument → Layer[16] → Voice — up to 16 stacked layers, each a polyphonic voice pool over its own patch graph.
  • 31 voice modules (buildSynthRegistry) — virtual-analog, wavetable, FM, additive, supersaw, Karplus and sampler oscillators; filters (SynthFilter / diode ladder); ADSR / LFO / S&H modulators; VCA / mixer / scaler.
  • FX-bus rack — a master bus plus 2 aux buses, each a patch graph running the full shared 340+-module audio registry (the same pack as zpwr-fx, including the 90+ analog models).
  • Per-param mod matrix, sidechain + global mods into the bus rack, and the shared zpc::WebEditor WebView UI.
