Engine

  • Instrument → Layer[16] → Voice on the shared zpc::LayeredEngine (zpwr-patch-core); each voice is a free patch graph.
  • 299 voice modules (buildSynthRegistry) — VA / wavetable / FM / additive / supersaw / Karplus / sampler oscillators, SynthFilter + diode ladder, ADSR / LFO / S&H, VCA / mixer / scaler.
  • FX-bus rack: master + 2 aux buses, each a patch graph running the shared 1190+-module audio registry (same pack as zpwr-fx, incl. 171 analog models).
  • Polyphonic voice pool with round-robin voice stealing; per-param mod matrix; sidechain + global mods into the bus rack.
