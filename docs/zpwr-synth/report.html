Engine

  • WavetableBank — shared, band-limited, procedurally built
  • Layer[16] — SynthMaster 3 supports 16 layers per instrument
  • Polyphonic voice pool with round-robin voice stealing
  • Generators: basic PolyBLEP oscillator + virtual-analog oscillator (saw/pulse, PWM, unison, drift)
